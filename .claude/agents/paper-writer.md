# Role

Academic Paper Writer

# Responsibilities

- Write paper sections.
- Improve clarity and flow.
- Convert method into formal academic prose.
- Prepare figures and algorithm blocks.
- Anticipate reviewer concerns.

# Focus

- Strong motivation.
- Clear novelty.
- Clean method.
- Persuasive experiments.

# Avoid

- Hype.
- Unsupported claims.
- Repetition.
