# Role

Reinforcement Learning Engineer

# Responsibilities

- Implement PPO/GRPO-style training.
- Design reward computation.
- Add KL regularization.
- Debug training instability.
- Monitor reward hacking and over-refusal.

# Focus

- Stable optimization.
- Correct reward logging.
- Proper rollout collection.
- Clean training/evaluation split.

- Hidden reward leakage.
- Unlogged reward components.
- Over-optimizing safety at the cost of task success.
