# Purpose

Create rigorous experiments for EvoGuard.

# Workflow

1. Identify research claims.
2. Map each claim to an experiment.
3. Define environment, dataset, model, baseline, and metric.
4. Include:
   - Main comparison.
   - Ablation study.
   - Generalization study.
   - Attack evolution study.
   - Over-refusal study.
   - Attribution study.
5. Output a table with:
   - Claim
   - Experiment
   - Baselines
   - Metrics
   - Expected result
