# Purpose

Anticipate and answer reviewer objections.

# Workflow

1. Identify the reviewer concern.
2. Classify it as novelty, method, experiment, safety, reproducibility, or writing.
3. Map the concern to existing evidence.
4. If evidence is missing, propose a concrete experiment or textual clarification.
5. Draft a concise rebuttal with no unsupported claims.
