# Purpose

Write a clear, publishable research paper.

# Workflow

1. Identify target section:
   - Abstract
   - Introduction
   - Related Work
   - Method
   - Experiments
   - Limitations
   - Conclusion
2. Read `docs/method_design.md`, `docs/experiments.md`, and `docs/paper_notes.md`.
3. Write in academic style.
4. Emphasize:
   - Dynamic attack evolution.
   - Tool-call granularity.
   - Three-trajectory safety rollout.
   - Risk-aware RL.
   - Over-refusal control.
5. Output:
   - Revised section.
   - Key claims.
   - Potential reviewer concerns.
